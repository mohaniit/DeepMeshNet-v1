# DeepMeshNet-v1

**Engineering Knowledge First. AI Second.**

DeepMeshNet-v1 is an explainable engineering AI framework for automated CAD mesh-density prediction.

The framework analyzes CAD geometry, applies explicit Engineering Mesh Knowledge, generates explainable mesh-density labels, and trains machine learning and graph neural network models to generalize those engineering decisions to unseen CAD models.

## Core Philosophy

Geometry provides facts.  
Engineering provides decisions.  
Artificial Intelligence learns those decisions.  
Visualization validates those decisions.

## Main Components

- `geometry` — CAD geometry analysis
- `knowledge` — Engineering Mesh Knowledge Base and Engine
- `graph` — CAD graph construction
- `learning` — classical ML and GNN models
- `visualization` — CAD rendering and validation
- `publication` — paper-ready figures, tables, and reports
- `utils` — shared utilities
